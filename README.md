# SCT_WD_3

## Project Overview  
SCT_WD_3 is a web development project built during my internship at SkillCraft Technology. It presents an interactive Tic Tak Toe game created using HTML, CSS, and JavaScript.

## Why this project?  
- To improve my front-end skills by designing interactive UI elements and application logic.  
- To build a deployable web game that showcases clean code structure and engaging UX.  
- To consolidate knowledge of HTML , CSS , JAVASCRIPT.

## Built With  
- HTML
- CSS (responsive layout, interactive styling)  
- JavaScript   
- Folder structure:  
  - `css/` – stylesheets  
  - `js/` – JavaScript files  
  - `assets/` – gif
  - `index.html` – entry point  

## Features  
- A game with interactive gameplay logic.   
- Visual indicators for correct/incorrect game states.  
- Responsive design to support both mobile and desktop viewing.  
- Clean, minimal dependency implementation for easy understanding and extension.

## Getting Started  

### Prerequisites  
- A modern web browser (Chrome, Firefox, Edge, Safari).  
- No special server setup required; you can open `index.html` locally.
